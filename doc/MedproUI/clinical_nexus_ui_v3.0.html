# Clinical Nexus UI 设计规范与页面索引 (V3.0) ## 1. 视觉系统设计 (Design System) - **品牌名称**：Clinical Nexus / 临床精准。 - **核心色彩**：医疗蓝 (#0056B3) 作为主色，辅以深蓝、浅灰及功能性色彩（成功绿、预警橙）。 - **字体规范**：标题使用 Manrope（加粗），正文使用 Inter/Source Han Sans，确保医学术语的清晰易读。 - **组件风格**：圆角采用 ROUND_FOUR，风格严谨、专业，具有高透明度的现代科技感。 ## 2. 导航架构规范 ### 2.1 顶部导航 (Top Global Nav) - **项**：首页、实验大厅、课程中心、资源中心、数据仪表盘。 - **交互**：当前页面高亮并带有底部蓝色粗下划线。右侧包含搜索、通知及个人中心。 ### 2.2 左侧树形导航 (Side Nav) - **教师端**：工作台、报告批阅、实验管理（虚拟/线下）、课程建设（编辑器/任务发布）、学情监控（班级/学生）、素材库。 - **学生端**：工作台、实验大厅、我的课程（学习/任务）、实验报告（撰写/成绩）、学情档案。 ## 3. 已定稿页面索引 ### 3.1 门户层 (访客/公共) - **首页**：{{DATA:SCREEN:SCREEN_61}} (优化排版版) - **实验大厅**：{{DATA:SCREEN:SCREEN_112}} (导航优化版) - **课程中心**：{{DATA:SCREEN:SCREEN_85}} (汉化聚合页) - **资源中心**：{{DATA:SCREEN:SCREEN_56}} (课程资源访客版) - **详情页**：单个课程 ({{DATA:SCREEN:SCREEN_51}})、仿真实验 ({{DATA:SCREEN:SCREEN_88}})、视频查看 ({{DATA:SCREEN:SCREEN_136}})。 ### 3.2 教师端 (管理/授课) - **控制台**：{{DATA:SCREEN:SCREEN_132}} (功能增强版) - **课程管理**：课程列表 ({{DATA:SCREEN:SCREEN_49}})、任务发布 ({{DATA:SCREEN:SCREEN_71}})、沉浸式授课 ({{DATA:SCREEN:SCREEN_119}})。 - **报告批阅**：任务清单 ({{DATA:SCREEN:SCREEN_40}})、学生清单 ({{DATA:SCREEN:SCREEN_13}})、批改详情 ({{DATA:SCREEN:SCREEN_38}})。 - **实验管理**：虚拟仿真 ({{DATA:SCREEN:SCREEN_142}})、线下管理 ({{DATA:SCREEN:SCREEN_121}})。 - **学情统计**：班级看板 ({{DATA:SCREEN:SCREEN_141}})。 ### 3.3 学生端 (学习/反馈) - **工作台**：{{DATA:SCREEN:SCREEN_74}} (最终优化版) - **我的课程**：课程清单 ({{DATA:SCREEN:SCREEN_17}})、任务进度 ({{DATA:SCREEN:SCREEN_62}})、学习详情 ({{DATA:SCREEN:SCREEN_72}})。 - **实验报告**：任务列表 ({{DATA:SCREEN:SCREEN_19}})、成绩查询 ({{DATA:SCREEN:SCREEN_63}})、在线撰写 ({{DATA:SCREEN:SCREEN_140}})。 ### 3.4 系统管理 - **组织管理**：{{DATA:SCREEN:SCREEN_131}} (姓名格式优化版) - **数据仪表盘**：{{DATA:SCREEN:SCREEN_114}} (中文大屏版) - **身份认证**：登录 ({{DATA:SCREEN:SCREEN_8}})、注册 ({{DATA:SCREEN:SCREEN_60}})。 ## 4. 关键设计模式 - **沉浸式授课**：移除全局导航，最大化教学内容展示空间。 - **双窗格批阅**：报告原文与评分表并列，减少视线切换。 - **自检预报**：通过不同色彩的图标（绿色/红色）预告实验环境是否达标。
