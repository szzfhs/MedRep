应用描述

本交互应用基于高精度CT与MRI数据重建，旨在为医学生提供一个沉浸式的人体结构学习平台。系统涵盖了骨骼、肌肉、循环、神经、消化及呼吸等十大系统，支持任意角度的缩放、旋转与解剖层级剥离。

通过实时物理引擎模拟，学生可以观察到心脏搏动、血液流动等动态生理过程。应用内置了超过3000个精确命名的解剖学结构，并配有专业的中英文注解与三维空间位置解析。
